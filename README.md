# Менеджер задач

В проекте реализованы 4 сущности: 
- Пользователь (может быть как автором задач, так и исполнителем)
- Задача
- Статус задачи
- Метка

## Требования

- PHP >= 8.1
- Composer >= 2
- make >= 4

## Установка
- make install

## Запуск приложения на localhost:8000
- make start

## Запустить тесты
- make test
